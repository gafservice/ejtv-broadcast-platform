---

# 4. Ubicación de la instalación

La estructura oficial de MediaMTX dentro de la plataforma EJTV es la siguiente:

```text
/opt/ejtv/mediamtx/
```

Organización de directorios:

```text
/opt/ejtv/mediamtx/

├── backup/
├── bin/
│   └── mediamtx
├── config/
│   └── mediamtx.yml
├── logs/
├── recordings/
├── releases/
├── scripts/
└── tests/
```

Esta organización separa claramente el software ejecutable, la configuración, los registros operativos y las versiones descargadas, facilitando el mantenimiento y futuras actualizaciones.

---

# 5. Configuración

Archivo principal de configuración:

```text
/opt/ejtv/mediamtx/config/mediamtx.yml
```

Versión del archivo:

```
v1.19.0
```

Durante esta línea base se mantiene la configuración oficial distribuida por MediaMTX, sin modificaciones funcionales.

Las personalizaciones específicas de la plataforma EJTV serán documentadas en futuras líneas base.

---

# 6. Servicio del sistema

MediaMTX fue integrado como un servicio nativo de **systemd**.

Archivo de servicio:

```text
/etc/systemd/system/mediamtx.service
```

Configuración principal:

* Inicio automático durante el arranque.
* Ejecución bajo el usuario `ejtv`.
* Reinicio automático en caso de fallo (`Restart=on-failure`).
* Registro de eventos mediante `journalctl`.
* Dependencia de `network-online.target`.

Estado operativo al cierre de esta línea base:

| Parámetro           | Estado |
| ------------------- | ------ |
| Servicio instalado  | Sí     |
| Servicio habilitado | Sí     |
| Servicio iniciado   | Sí     |
| Inicio automático   | Sí     |
