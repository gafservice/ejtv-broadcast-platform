# BASELINE-013
# Secure Reliable Transport (SRT)

**Proyecto:** EJTV Broadcast Platform

**Misión:** MISSION-013 — SRT

**Estado:** FINAL

---

# Resumen

La plataforma incorpora soporte completo para el protocolo Secure Reliable Transport (SRT) mediante MediaMTX, permitiendo la publicación y lectura de flujos audiovisuales sobre UDP con baja latencia.

La implementación fue validada mediante pruebas funcionales utilizando FFmpeg y FFprobe.

# Configuración MediaMTX

Archivo:

```text
/opt/ejtv/mediamtx/config/mediamtx.yml
```

Configuración aplicada:

```yaml
srt: true

srtAddress: :8890
```

Parámetros disponibles:

```yaml
srtReadPassphrase:

srtPublishPassphrase:
```

No fueron configuradas contraseñas durante esta misión.
