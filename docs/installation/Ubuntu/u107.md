# Parte 4.7 — U-007 Establecimiento de la Línea Base del Servidor (Baseline BL-001)

---

# Introducción

La instalación de un sistema operativo constituye únicamente la primera etapa
en la construcción de un servidor.

Una vez finalizada la instalación de Ubuntu y completada la caracterización del
hardware, el almacenamiento, la infraestructura de red y el estado operativo
del sistema, resulta necesario consolidar toda esta información en una única
referencia técnica.

En ingeniería de infraestructura este procedimiento se conoce como
**establecimiento de una línea base** (*Baseline*).

Una línea base representa el estado oficial del sistema en un momento
determinado y constituye el punto de referencia contra el cual se compararán
todas las modificaciones futuras.

La creación de esta línea base permite mantener la trazabilidad del proyecto,
facilita la administración del servidor y simplifica los procesos de auditoría,
mantenimiento y recuperación.

Para **EJTV Broadcast Platform**, esta línea base corresponde al estado del
servidor **ejtv-01** inmediatamente después de concluir la instalación de
Ubuntu 24.04.4 LTS y antes de incorporar cualquier componente propio del
proyecto.

---

Establecer el estado técnico oficial del servidor **ejtv-01**, consolidando la
información obtenida durante la caracterización inicial y definiendo el punto

La instalación del sistema operativo fue realizada correctamente, el hardware

necesaria durante las primeras etapas de desarrollo, permitiendo planificar una

# Implicaciones para el proyecto

La línea base BL-001 representa el punto de partida oficial de
**EJTV Broadcast Platform**.

A partir de este momento, toda modificación realizada sobre el servidor deberá
quedar documentada y podrá compararse con esta referencia inicial.

Esta metodología permitirá reconstruir históricamente la evolución completa de
la infraestructura, facilitando las tareas de mantenimiento, auditoría y
gestión de configuración.

Además, la existencia de una línea base técnica claramente documentada reduce
la incertidumbre durante futuras actualizaciones del sistema operativo, cambios
de hardware o incorporación de nuevos servicios.
